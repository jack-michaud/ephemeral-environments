# bd Command Reference

For detailed flags and options on any bd command, run:

```bash
bd <command> --help
```

## Commonly Used Commands

| Command | Purpose |
|---------|---------|
| `bd create --help` | Creating issues with all available flags |
| `bd list --help` | Filtering and listing issues |
| `bd update --help` | Modifying existing issues |
| `bd show --help` | Viewing issue details |
| `bd close --help` | Closing issues |
| `bd dep --help` | Managing dependencies |
| `bd graph --help` | Visualizing dependencies |

## Discovery Commands

```bash
bd --help           # All available commands
bd human            # Essential commands for human users
bd prime            # AI-optimized workflow context
bd quickstart       # Quick start guide
```

## Getting Current State

```bash
bd info             # Database and daemon information
bd status           # Issue counts and overview
bd doctor           # Check installation health
```
